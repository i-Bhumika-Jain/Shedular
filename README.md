# Daily Planner Application

This is a React application built with Vite.

## How to Run the Application

To start the development server, follow these steps:

1. Open your terminal.
2. Navigate to your mapped drive project directory instead of the network path (to avoid path issues):
   ```cmd
   H:
   cd H:\Desktop\TT
   ```
3. Run the development script:
   ```cmd
   npm run dev
   ```
4. Open the local URL provided in the terminal (usually `http://localhost:3000`) in your web browser.

## Other Commands

- **Build for production:**
  ```cmd
  npm run build
  ```
- **Preview production build locally:**
  ```cmd
  npm run preview
  ```
